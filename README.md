# הוראות לעדכון התמונות (How to update images)

האתר מוכן! כרגע הוא מציג תמונות דוגמה (Placeholders). כדי להציג את התמונות של המשפחה שלך:

1.  גש לתקייה `images` בתוך הפרוייקט.
2.  החלף את הקבצים `photo1.png`, `photo2.png`, `photo3.png` בתמונות שלך.
    *   **חשוב**: שמור על אותם שמות קבצים בדיוק (`photo1.png` וכו'), או שנה את הסיומת ב-`index.html` אם התמונות שלך הן `.jpg` או `.webp`.
    *   מומלץ לדחוס את התמונות כדי שהאתר יטען מהר (למשל באתר [tinypng.com](https://tinypng.com)).
3.  אם אתה רוצה להוסיף עוד תמונות:
    *   הוסף את התמונה לתקייה `images`.
    *   פתח את `index.html`.
    *   העתק את אחד הבלוקים של `photo-card` והדבק אותו מחדש.
    *   שנה את ה-`src` לשם הקובץ החדש.
    *   שנה את המחלקה (Class) ל-`pos-4` (או מספר אחר) והוסף עיצוב מתאים ב-`style.css` אם תרצה מיקום ייחודי.

תהנה!
